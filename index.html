Git & GitHub-Complete Guide

  • Git aur GitHub version control system aur collaboration tool hai jo developers ko code manage karne aur track karne me help karta hai.
  • Chalo Git aur GitHub ko basic se advanced level tak commands ke saath samajhte hain.

Git Kya hai??

  • Git ek distributed version control system hai jo code changes track karta hai aur multiple developers ko team me milke kaam karne me help karta hai.
  • Agar Git nahi hota, to har baar naye changes ko manually save aur share karna padta, jo kaafi problem create karta hai.
  • Git ka use karne se code ka pura history track hota hai aur kabhi bhi previous version pe wapas ja sakte hain!

Example:

Agar tumhari team me 5 log ek project pe kaam kar rahe hain, to sab apni apni changes ko Git me track kar sakte hain bina kisi conflict ke.

GitHub Kya Hai?

Isme tum apne Git repositories ko store, share aur manage kar sakte ho.
Git aur GitHub ek dusre se alag hain:
Git → Local system pe version control ke liye.
GitHub → Cloud me repositories ko store aur manage karne ke liye.

Git Installation & Setup

Step 1: Install Github

Windows: Download Git
Mac: brew install git
Linux: sudo apt install git

Step 2: Git Configure Karo

Git setup karne ke liye user name aur email set karna padta hai, jo commits ke saath save hota hai.

git config --global user.name "Tumhara user Name GitHub wala yaha likho"
git config --global user.email "tumhara GitHub Wala email@example.com yaha likho"

Check Configuration:

git config --list

Git Basic Commands

New Git Repository Initialize Karna

git init

✅ Ye command ek naya Git repository create karti hai.

Git Repository Ko Status Check Karna

git status

✅ Ye command batati hai ki kya changes stage ho chuke hain ya nahi.

Files Ko Staging Area Me Add Karna

git add filename # Single file add karne ke liye
git add . # Saari files ek sath add karne ke liye (add or dot ke bich space rkhna jarur)

✅ Staging area wo place hai jaha files commit hone se pehle ready hoti hain.

Changes Ko Commit Karna

git commit -m "Commit message yaha likho"

✅ Ye command staged changes ko repository me save karti hai.

Git Branching (Feature Development Ke Liye)

Branch ka use alag-alag features ya bug fixes ke liye kiya jata hai bina main code ko effect kiye.

Naya Branch Banana

git branch new-branch-name

✅ Ye command naya branch create karti hai.

Branch Switch Karna

git checkout new-branch-name

✅ Ye command tumhe dusre branch me shift karne me help karti hai.

git checkout -b new-branch-name

✅ Ye command tumhe new branch create krke us branch me shift karne me help karti hai.

Git Merge (Branches Ko Merge Karna)

git checkout main
git merge new-branch-name

✅ Ye command kisi branch ke changes ko main branch me merge karti hai.

Branch Ko Delete Karna

git branch -d new-branch-name

✅ Ye command branch delete karti hai agar wo merge ho chuka ho.

GitHub Ke Sath Kaam Karna

Git Repository Ko GitHub Se Connect Karna

git remote add origin https://github.com/username/repository.git

✅ Ye command local Git repository ko GitHub repository se connect karti hai.

Git repositories Ko dekhna

git remote -v

✅ Ye command remote Git repositories ko dikhati hai.

Code Ko GitHub Pe Upload Karna (Push)

git push -u origin main

✅ Ye command tumhare local commits ko GitHub pe upload karti hai.

GitHub Se Code Download Karna (Clone)

git clone https://github.com/username/repository.git

✅ Ye command GitHub ka repository copy karke local machine pe download karti hai.

Dusre Developers Ke Changes Ko Update Karna (Pull)

git pull origin main

✅ Ye command GitHub se naye changes ko local repository me update karti hai.

Advanced Git Commands

Git Log (History Check Karna)

git log

✅ Ye command poora commit history dikhati hai.

git log --oneline #commit history ko oneline me dekhna
git log --graph --decorate --all # history ko graph ke roop me dikhana

Git Reset (Last Commit Ko Undo Karna)

git reset --hard HEAD~1

✅ Ye command last commit ko hata deta hai aur previous state pe le jata hai.

Git Stash (Temporary Changes Ko Store Karna)

git stash
git stash pop

✅ Agar tum changes ko temporarily save karna chahte ho, to git stash use karo aur baad me git stash pop se wapas le aao.

git stash list #stashed changes ko dekhne ke Liye.
git stash apply # reapply stashed changes.
git stash drop #stashed changes ko hatane ke Liye.

Git Fetch (Remote Changes Ko Check Karna)

git fetch

✅ Ye command remote changes ko check karti hai.

Git Rebase (Branches Ko Clean Karna)

git rebase main

✅ Ye command current branch ko clean karti hai aur main branch ke saath merge karti hai.

Git Cherry-Pick (Specific Commits Ko Copy Karna)

git cherry-pick commit-id

✅ Ye command specific commit ko copy karke current branch me paste karti hai.

Git Tag (Specific Version Ko Mark Karna)

git tag -a v1.0 -m "Version 1.0"

✅ Ye command specific version ko mark karti hai.

git tag

✅ Ye command tags ko show karti hai.

git show v1.0

✅ Ye command specific tag ko show karti hai.

Git Submodule (External Repositories Ko Add Karna)

git submodule add repository-url

✅ Ye command external repositories ko add karti hai.

git submodule update --init --recursive

✅ Ye command submodules ko update karti hai.

Git Bisect (Bug Find Karna)

git bisect start
git bisect bad
git bisect good commit-id

✅ Ye command bug find karti hai.

Git Blame (Code Ka Author Check Karna)

git blame filename

✅ Ye command code ka author check karti hai.

Git Archive (Repository Ko Zip Me Convert Karna)

git archive -o filename.zip main

✅ Ye command repository ko zip me convert karti hai.

Git Worktree (Multiple Branches Ko Manage Karna)

git worktree add branch-name

✅ Ye command multiple branches ko manage karti hai.

Git LFS (Large Files Ko Track Karna)

git lfs install
git lfs track "*.psd"
git add .gitattributes
git commit -m "Add LFS"

✅ Ye command large files ko track karti hai.

Git Revert (Commit Ko Undo Karna Without Deleting History)

git revert commit-id

✅ Agar kisi commit ko undo karna ho bina delete kiye, to git revert use karo.

Git Reflog (Reference Logs Check Karna)

git reflog

✅ Ye command reference logs ko check karti hai.

Git Hooks (Automate Tasks)

.git/hooks

✅ Ye command automate tasks ko run karti hai.

Git Diff (Changes Check Karna)

git diff

✅ Ye command changes ko check karti hai.

Git Ignore (Files Ko Ignore Karna)

echo "file_name extention ke sath" >> .gitignore

✅ Ye command files ko ignore karti hai.

git clean -f # untracked file ko hata dega

Git Aliases (Commands Ko Shortcuts Me Convert Karna)

git config --global alias.co checkout
git config --global alias.br branch
git config --global alias.ci commit
git config --global alias.st status

✅ Ye command commands ko shortcuts me convert karti hai.

Git & GitHub Ka Real-Life Use Case

  • Tumhari team ek website ka project develop kar rahi hai.
  • Sab apne features alag-alag branches me add kar rahe hain.
  • Jab feature complete hota hai, to changes ko GitHub pe push karte hain.
  • Team leader changes ko review karke merge kar leta hai.
  • Agar koi problem aati hai, to previous version restore kar sakte hain.
